AST Parsing

Python's native ast module converts raw source into a structural tree graph. Recursive traversal computes nesting depth, counts branches, and flags dangerous nodes like eval().

ast.walk() · ast.NodeVisitor
◈
Cyclomatic Complexity

McCabe complexity approximation counts all branch nodes — If, For, While, ExceptHandler — plus boolean operators. Produces an A–F grade scale.

McCabe Score · Risk Index
⬢
Async Queue (Celery + Redis)

LLM API calls are decoupled from the Django request cycle. Text jobs are pushed into Redis, returning 202 Accepted instantly while Celery workers process in the background.

Non-blocking · 202 Accepted
◉
Perplexity + Burstiness

Pre-filters strip AI fingerprints via regex. Burstiness (coefficient of variation of sentence lengths) and perplexity (type-token ratio) are computed before and after transformation.

Regex Pipeline · Linguistic Metrics
⬛
Atomic DB Transactions

Every submission is wrapped in transaction.atomic(). If either analysis track fails mid-process, PostgreSQL rolls back — no partial, corrupted records survive.

PostgreSQL · ACID Compliant
∑
SQL Aggregations

Dashboard statistics use raw AVG(), SUM(), and string-difference aggregations computed inside the database — a single compiled row, not thousands fetched to Python.

In-DB Compute · Single Row Response
